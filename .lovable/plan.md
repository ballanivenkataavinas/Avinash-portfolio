

# Cybersecurity Portfolio Website

## Overview
A dark-themed, professional cybersecurity portfolio website with a hacker/security engineer aesthetic — deep blacks and blues, subtle neon glow effects, smooth scroll animations, and clean typography.

## Pages & Sections (Single Page with Smooth Scroll)

### 1. Hero Section
- Full-screen dark hero with subtle animated grid/matrix background effect
- Name, title, tagline with glowing text accents
- Professional intro paragraph
- Two CTA buttons: "View Projects" and "Contact Me" with glow hover effects

### 2. About Me
- Professional summary with a terminal/code-style accent element
- Clean two-column layout with text and a subtle decorative element

### 3. Technical Skills
- Categorized skill cards with icons (Security Tools, Backend, Database, Concepts, Virtualization)
- Subtle hover glow effects on skill badges

### 4. Featured Projects
- Four detailed project cards in a professional format
- Each card shows: Problem → Approach → Tools → Security Impact
- Expandable or stacked card layout with tool badges

### 5. Certifications & Learning
- Current learning tracks displayed as progress items
- Placeholder slots for future certifications

### 6. Career Objective
- Clean statement section with accent styling

### 7. Contact Section
- Email, LinkedIn, GitHub links with icons
- Simple contact form or direct links
- Footer with copyright

## Design Details
- **Color palette**: Deep black (#0a0a0f), dark blue (#0d1117), cyan/teal accents (#00d4ff), subtle green (#00ff88)
- **Typography**: Clean sans-serif (Inter/system fonts)
- **Animations**: Fade-in on scroll, subtle glow pulses, hover transitions
- **Fully responsive** for mobile, tablet, and desktop
